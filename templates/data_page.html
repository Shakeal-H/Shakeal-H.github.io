About the Data

General Information

This Pokedex dataset includes a Pokemon’s name and stats (hp, attack, defense, special attack, special defense, speed, type, height, weight, weaknesses, strengths, catch rate, base friendliness, etc) and what region the Pokemon are from. Our data was accessed on January 21st, 2022 from Kaggle and contains data regarding various stats. We then added a column containing images to the dataset so that we could display them for individual Pokemon. These images were found on Bulbapedia, and they are free to the public to use.

How was the Data Collected?

As stated on Kaggle, “The data was collected by data mining (with a Python Web crawler) the following websites: https://pokemondb.net/ (almost everything) https://www.serebii.net/ (information about sub-legendary, legendary and mythical Pokemon) The pictures for this Dataset were found in the Wikipedia.” The data was last updated on 2021-04-16, and it contains information on all Pokemon through generation 8.
